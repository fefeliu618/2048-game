Game Goal

策略与节奏

尽量让大数字留在角落，优先保持棋盘上有足够空位。每一步都兼顾合并收益与后续可移动空间，能走得更远。

Input Design

双端操作一致

页面基于拖拽与滑动手势实现方向控制，桌面端使用鼠标拖动，移动端使用触屏滑动；同时也保留键盘方向键支持，方便补充操作。

Local Feature

轻量但完整

支持撤销一步、操作提示、胜利与失败状态弹层，以及本地最高分记忆。无需任何依赖，直接打开页面即可开始游戏。
